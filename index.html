Supergiant Games

Return to Top

This is all work done as mods or tools to the games Pyre, Hades, and Hades 2 developed and produced by Supergiant Games (https://www.supergiantgames.com)

All menus shown here were made utilizing only resources and images found within Supergiant Game's games Pyre, Hades, and Hades 2:

UI

Pyre in Hades

An example mod for creating new maps included this UI. Created mostly by utilizing images from Pyre.

UI Library

A library for both Hades 1 and Hades 2 implementing many different UI components, utilized by the Hades 1 speedrunning community

Challenge Mod

A mod that interfaces with PonyWarrior's challenge mod implemented this UI.

Accessibility Mods

Mods for both Hades 1 and Hades 2 which overhaul many menus and aspects of the game to allow blind or visually impaired players to play the game.

Made with support and feedback of players at https://forum.audiogames.net/.

Hades 2 accessibility features are in active development and include a built-in text-to-speech system which automatically interfaces with the menus in-game.

Map Editing

I reverse engineered Supergiant's proprietary map binaries allowing compilation and decompilation. This allows adding new maps (as seen above) or editing existing ones.
